# 00 — Обзор курса

## Что за курс

**AI Advent Challenge #8** — Алексей Гладков, mobiledeveloper.tech.
7 недель практики по разработке с ИИ.
Страница курса: https://mobiledeveloper.tech/ai_advent_8 (может быть за авторизацией).

Этот репозиторий — рабочий журнал: код заданий, конспекты, прогресс.

## Правила челленджа

- Задания выходят **пн–пт в 14:00 МСК**. Всего 7 недель.
- Формат сдачи каждого задания: **видео + код**, причём **видео важнее кода**.
- Результаты вношу в общую таблицу курса.
- **Не сдал 5 заданий за неделю → выбываю.**
- Опоздание по сдаче — максимум **15 минут**.
- Принцип курса: всё делаем **через ИИ, код руками не пишем**.
- VPS понадобится **с 6-й недели**.
