# Week 07 — растущий dev-ассистент

Дни 31–35. Целевой проект — `agent-web` (этот репозиторий) само по себе: RAG-ассистент по
собственному коду/докам вместо внешнего Next.js/GitVerse проекта (см. "Что изменилось"
ниже — вопросы из старой версии этого README закрыты именно так). Пять дней подряд один
и тот же чат-агент обрастает возможностями по модулям, не пять отдельных приложений.

## Что изменилось относительно исходного PLAN-BRIEF

Старая версия этого README (до дня 31) держала блок "Открытые вопросы", предполагавший
целевым проектом внешний Next.js + TS + Postgres + S3 на GitVerse — доступ к нему решили
не подключать. Реально исполненный план (`swarm-report/week07-dev-assistant-plan.md`,
раздел "День 35" и выше) выбрал целевым проектом сам `agent-web`/`agent-cli` — RAG-индекс
строится из собственного кода/доков (`build_project_corpus.py`), MCP-сервер (день 31)
отдаёт git-статус/диффы этого же репо, AI-ревью (день 32) ревьюит PR этого же репо,
саппорт (день 33) отвечает по тикетам из реальных багов недель 6-7, файловый агент
(день 34) читает/пишет файлы этого же репо в песочнице. День 35 (инвест-бот как кандидат
из старой версии README — отклонён в PLAN-BRIEF) стал автоматизацией самого курса:
ритуал записи прогресса в README/progress.md.
